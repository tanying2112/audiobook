# 行为准则 (Code of Conduct)

## 我们的承诺
本项目致力于为所有贡献者提供一个安全、友好、包容的环境。我们不容忍任何形式的骚扰、歧视或不尊重行为。

## 可接受的行为
- 尊重他人的不同背景、经验和观点。
- 使用友好、专业的语言进行沟通。
- 对所有贡献者的代码、文档和意见给予建设性的反馈。

## 不可接受的行为
- 任何形式的性别、种族、宗教、年龄、性取向、残疾或其他身份的歧视。
- 人身攻击、侮辱、威胁或骚扰。
- 在公开或私下渠道发布不适当的内容（包括但不限于淫秽、暴力、仇恨言论）。

## 违规处理
如果您在项目中遇到或目睹上述行为，请通过以下方式报告：
- 在 GitHub Issues 中打开 **Private** Issue（仅限项目维护者可见），或
- 发送邮件至项目维护者（在 `README.md` 中提供联系方式）。

项目维护者将在收到报告后尽快调查，并根据情况采取适当的措施，包括但不限于警告、删除评论、封禁账户等。

## 参考
本行为准则基于 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。
